# Calliope Releases

This repository hosts release artifacts and the auto-update manifest for [Calliope](https://github.com/LukeGeneva/calliope), a writing app for authors.

## Downloads

Find the latest installer for your platform on the [Releases](https://github.com/LukeGeneva/calliope-releases/releases) page.

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `Calliope_x.x.x_aarch64.dmg` |
| macOS (Intel) | `Calliope_x.x.x_x64.dmg` |
| Windows | `Calliope_x.x.x_x64-setup.exe` |

## Auto-updates

Calliope checks for updates automatically on launch. When a new version is available, you'll see a prompt in the app to install and restart.

The update manifest is served from [`latest.json`](./latest.json) in this repository.

## Source Code

The source code is maintained in a private repository. This repo exists solely to distribute builds to testers.
